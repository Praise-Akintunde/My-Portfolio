
# Akintunde Praise Oreoluwa - Portfolio
A modern, interactive one-page portfolio website for Akintunde Praise Oreoluwa showcasing skills, experience, and projects.

## Features
- Dark/Light mode toggle
- Smooth scroll navigation
- Animated sections: Hero, About, Skills, Experience (timeline), Projects, Contact
- CV download button
- Responsive design

## Technologies Used
- HTML
- CSS
- JavaScript

## How to View
Just open `index.html` in your browser!
